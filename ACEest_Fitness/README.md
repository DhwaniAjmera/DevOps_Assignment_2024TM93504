# ACEest Fitness & Gym - DevOps Assignment

## Project Overview
This project implements a Flask web application for managing ACEest Fitness & Gym members.  
It demonstrates a full **DevOps CI/CD pipeline** using GitHub Actions, Jenkins, Docker, and unit testing with Pytest.
